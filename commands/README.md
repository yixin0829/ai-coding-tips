# Commands (aka Workflows or Prompt Templates)

Commands are used to automate repetitive complex tasks. It's basically prompt templates branded differently. You may put your commands in the `.cursor/commands/` folder or `.claude/commands/` folder or `.windsurf/workflows/` folder.

## Location
- `.cursor/commands/` for Cursor
- `.claude/commands/` for Claude Code
- `.windsurf/workflows/` for Windsurf


Ensure to toggle on "Import Claude Commands" in Cursor's "Rules and Commands" settings. Cursor will then automatically pick up the commands in both folders.
